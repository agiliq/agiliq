what we do

We work with some of the most cutting edge technologies, open web has to offer. Building on best of breed open technologies allows us to be fast, secure, scalable.

python

Python is an easy to use but mature language with strong community and open source roots. It is used by some of the the largest corporations like Google and Nasa, and is yet liked by hobbyists.

django

Django bills itself as "the Web framework for perfectionists with deadlines." It has emerged as the most widely used Python based framework. Google used parts of Django for the App Engine.

Mysql and Postgresql

Mysql and postgresql are the two best open source database available. They have proved their ability at some of the largest websites, power Yahoo, Facebook and the like.

jquery

Jquery is a Javascript library designed to make DOM traversal, event handling and Ajax interaction easy. With Jquery unobtrusive Javascript is amazingly simple.

user interface design

The best technology is meaningless if it does not help the user interact with the website in an easy and consistent manner.

xhtml & css

High quality, cross-browser compatible valid XHTML/ CSS markup is what the best sites are built on and allow applications to reach the largest market available — every person's browser.
