On-site ProcessingÂ¶

Onsite processing refers to the payment mechanism where the customer stays on the merchant website and the authentication is done by the merchant website with the gateway in the background.

Merchant websites need to comply with PCI standards to be able to securely carry out transactions.

On-site processing for payment gateways is implemented by using subclasses of the Gateway class.
